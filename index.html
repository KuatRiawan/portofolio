Tentang Saya

Perjalanan Saya di Dunia Teknologi

  1. 2023 - Awal Perjalanan

    Memulai Studi

    Memulai studi S1 Sistem Informasi di Universitas Terbuka, membangun fondasi di bidang teknologi dan manajemen.

  2. 2024 - Pengembangan Keterampilan

    Eksplorasi Web & Data

  3. 2025 - Teknologi Modern

    API & Framework

    Mulai bereksperimen dengan API (Gemini) dan framework modern (Tailwind CSS) untuk membangun aplikasi yang lebih interaktif.

Sebagai mahasiswa Sistem Informasi dan juga Training Coordinator di perusahaan PJK3, saya banyak belajar bagaimana teknologi bisa membantu meningkatkan efisiensi kerja dan pengelolaan data. Bagi saya, memahami proses bisnis dan menghubungkannya dengan sistem digital adalah langkah penting untuk menciptakan solusi yang berdampak nyata di dunia kerja.

Saya tertarik mengembangkan sistem yang sederhana tapi fungsional, terutama yang bisa membantu otomatisasi tugas administrasi dan pelatihan. Saat ini, saya aktif memperdalam pengembangan web, analisis data, serta eksplorasi API dan framework modern untuk membangun sistem berbasis data yang lebih terintegrasi dan mudah digunakan.

  • Terbiasa berpikir analitis dan menyelesaikan masalah dengan pendekatan praktis
  • Belajar secara mandiri dan terus mencari cara baru untuk meningkatkan kemampuan
  • Berfokus pada efisiensi kerja dan penerapan teknologi dalam pengembangan SDM
